Context: A database containing template information, document information and paragraphes in documents.

Below are 2 good examples:
Question: Show ids for all templates not used by any document.
Good Paraphrase 1: What are the IDs for templates not associated with a document?
Good Paraphrase 2: which temples not used any document? what are the ids of those templates

Question: Return the lowest version number, along with its corresponding template type code.
Good Paraphrase 1: What is the earliest version and its template type code?
Good Paraphrase 2: what is the lowest version number as well as its corresponding model type code?

Below are a bad example:
Question: Return the lowest version number, along with its corresponding template type code.
Bad Paraphrase 1: What's the template type code with the lowest version number? (It is not a good paraphrase because the original question asks for lowest version number and template type code and the paraphrase question only asks to return the template type code)
Bad Paraphrase 2: What's the template type code with the lowest version number? (It is not a good paraphrase because template code doesn't mean the same thing as template type code)
